USAGE
=====


```
## COLORS
msg @red blabla
msg @lightgreen "lalalalalolo"
msg @black:orange "lalalalalolo"

## BOLD
msg _"lololo"_

## UNDERLINE
msg __"lololo"__

## BOLD +UNDERLINE
msg ___"lololo"___

## NEW LINE
msg \n lol \n

## TAB SPACE
msg \t lolol \t\t lala

## URL
msg [google.com]

## DIRECTORY
msg /home/
```
